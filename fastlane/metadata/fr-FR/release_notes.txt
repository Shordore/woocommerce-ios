Nous sommes heureux de vous présenter les dernières mises à jour de WooCommerce pour iOS. Vous pouvez désormais vous lancer dans la création de votre boutique immédiatement après avoir créé un nouveau compte WPCOM. Nous avons également amélioré la visibilité des informations relatives aux cartes-cadeaux sur plusieurs commandes. De plus, nous avons activé les fonctionnalités d’IA pour la description et le partage des produits pour les sites auto-hébergés avec l’extension IA Jetpack. Enfin, les cartes-cadeaux appliquées apparaîtront désormais sous la section Code promo sur le formulaire de commande afin de faciliter le suivi des remises. Profitez bien de ces améliorations !
